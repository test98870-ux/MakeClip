# 가정예배 모음 영상 제작 과정 보고서

본 보고서는 `MakeClip` 프로젝트의 결과물인 3분 분량의 가정예배 모음 영상 제작 과정을 상세히 설명합니다.

## 1. 개요
- **목적:** 교회 대예배 상영용 가정예배 하이라이트 영상 제작
- **대상:** 프로젝트 내 포함된 모든 가정 (6개 그룹)
- **도구:** Python, MoviePy, FFmpeg, OpenAI Whisper

## 2. 제작 단계

### 1단계: 동영상 분석 및 클립 선정
- `media/output` 폴더 내의 각 가정별 `summary.txt`를 정독하여 가장 은혜롭고 의미 있는 구간을 선정했습니다.
- **선정 기준:**
  - 아이들의 순수한 찬양 고백
  - 부모와 자녀가 함께 성경을 읽는 모습
  - 신앙적인 교훈을 나누는 대화 장면
  - 간절한 합심 기도와 마무리 "아멘"

### 2단계: 스토리보드 설계 (`STORYBOARD.md`)
- 영상의 흐름을 **오프닝 -> 찬양 -> 말씀 -> 나눔 -> 기도 -> 클로징**으로 구성하여 한 편의 짧은 예배를 드리는 듯한 느낌을 주었습니다.
- 각 가정의 분량을 균등하게 배분하여 공동체성을 강조했습니다.

### 3단계: 기술적 구현 (`assemble_video.py`)
- `MoviePy` 라이브러리를 사용하여 선정된 클립들을 자동으로 병합했습니다.
- **주요 처리 사항:**
  - **오디오 지원:** 영상이 없는 청소년 예배 음성(수혁)의 경우, 검은 배경을 생성하여 오디오를 삽입했습니다.
  - **해상도 통일:** 다양한 비율의 영상들을 세로 720p 기준으로 리사이징하여 시각적 이질감을 줄였습니다.
  - **안전한 슬라이싱:** 클립의 실제 길이를 체크하여 `ValueError`를 방지하는 방어적 코드를 적용했습니다.

### 4단계: 결과물 출력
- **경로:** `media/result/GeminiCLI/Worship_Gathering_3min.mp4`
- **형식:** H.264 (MP4), AAC 오디오

- **배경음악(BGM) 추가:** 현재는 현장음 위주로 구성되어 있으나, 배경에 잔잔한 피아노 반주를 깔면 더욱 은혜로운 분위기를 연출할 수 있습니다.
- **자막 작업:** 각 클립 하단에 가정 이름과 말씀 구절 자막을 삽입하면 성도들이 내용을 더 쉽게 파악할 수 있습니다.
- **전환 효과:** 현재는 단순 컷 편집 위주이나, 클립 사이사이에 '디졸브'나 '페이드' 효과를 주면 더욱 전문적인 영상 느낌을 줍니다.
