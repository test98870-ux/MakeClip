# Codex 제작 과정

## 작업 기준
`요구사항.md`를 기준으로 기존 분절 결과(`media/output`)를 재사용하고, 기존 `assemble_video.py`는 유지한 채 Codex 전용 조립 스크립트를 별도로 추가했다.

## 사용 파일
- 편집 스크립트: `assemble_video_codex.py`
- 스토리보드: `CODEX_STORYBOARD.md`
- 결과 폴더: `media/result/Codex`

실제 사용된 분절 파일 목록은 렌더링 후 `media/result/Codex/USED_FILES.md`에 기록되도록 했다.

## 편집 방식
- 찬양, 말씀봉독, 나눔, 기도 순으로 구간을 재배열했다.
- 다섯 가정과 청소년 음성이 모두 포함되도록 클립을 고정 선택했다.
- 영상은 720p 기준으로 통일하고, 오디오는 원본 현장음을 유지한다.
- 별도 배경음악은 추가하지 않고 원본 예배 음성 중심으로 구성했다.

## 추가 개선 방향
- 장면 전환을 더 부드럽게 하려면 crossfade와 볼륨 정규화를 넣는 편이 좋다.
- 자막을 넣으려면 가정명과 핵심 문장을 하단 자막으로 추가할 수 있다.
- 결과 품질을 높이려면 최종 선택 클립을 사람이 한 번 더 미세 조정하는 것이 안전하다.
